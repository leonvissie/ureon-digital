# Ureon Digital Website — Foundation Build v1

This is the initial local-development scaffold for the Ureon Digital website.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- npm

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Included

- Homepage foundation
- Navbar and footer
- Hero section
- Featured app cards
- Principle cards
- About/studio section
- Initial motion system
- Design tokens
- Placeholder app/legal pages
